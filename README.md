# Quake Feed App

## This is a sample application meant to display earthquake measurements based on zipcode

_____________________________________________________

### Getting Started

-----------------------------------------------------
* Start by cloning the repo at the following address: https://github.com/atadjc89/quake_app_practice.git

* Navigate to the root of the directory
* Open two parallel terminals
* In both of the terminals, cd into the `/react_sample` folder

### First Terminal

* In one of the terminals, cd into the `/server` folder
* run the command `npm start` to run the server and migration files

### Second Terminal

* In the other terminal, cd into the `/react-app` folder
* Enter the following command into the terminal:
```
npm run dev
```
* Click the link http://localhost:5174/ to open the webpage
